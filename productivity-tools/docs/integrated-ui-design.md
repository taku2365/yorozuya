## 3. 連携機能のUI

### 3.1 タスク詳細ダイアログ（共通）
```
┌─ タスク詳細 ────────────────────────────────────────────────────┐
│ タイトル: [_____________________]                              │
│                                                                │
│ [基本情報タブ] [関連タブ] [履歴タブ] [コメントタブ]              │
│ ┌──────────────────────────────────────────────────────────┐ │
│ │ 関連タブ                                                  │ │
│ │                                                           │ │
│ │ Todo表示:      ☑ 有効（期限: 2024-01-15）                │ │
│ │ WBS位置:       プロジェクトA > フェーズ1 > タスク1.1       │ │
│ │ カンバンレーン: 進行中                                     │ │
│ │ ガント表示:    ☑ 有効（1/10 - 1/15）                     │ │
│ │                                                           │ │
│ │ [他のビューに追加] [リンクを解除]                          │ │
│ └──────────────────────────────────────────────────────────┘ │
│                                     [キャンセル] [保存]        │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 クイックアクションメニュー
```
タスクを右クリック時:
┌─────────────────────┐
│ ✏️ 編集             │
│ 📝 Todoに追加       │
│ 🌳 WBSに追加        │
│ 📋 カンバンに追加    │
│ 📊 ガントに追加     │
│ ─────────────────  │
│ 📎 リンクをコピー    │
│ 🗑️ 削除            │
└─────────────────────┘
```

### 3.3 ドラッグ&ドロップ連携
- Todoからカンバンレーンへドラッグ → カンバンカード作成
- WBSからガントへドラッグ → ガントタスク作成
- カンバンからTodoへドラッグ → Todo項目作成

## 5. レスポンシブ対応

### モバイル表示（縦向き）
- ビュー切り替えは下部タブバー
- サイドバーはハンバーガーメニュー
- 各ビューはモバイル最適化表示

### タブレット表示
- サイドバーは折りたたみ可能
- ビュー切り替えは上部タブ維持
- 2カラムレイアウト対応

## 6. アクセシビリティ

- キーボードナビゲーション完全対応
- スクリーンリーダー対応
- ハイコントラストモード
- フォントサイズ調整機能

## 7. カスタマイズ機能

- ビューの表示/非表示設定
