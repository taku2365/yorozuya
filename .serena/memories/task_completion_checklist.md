# タスク完了時のチェックリスト

## 必須確認項目

### 1. テスト
```bash
npm test
```
- すべてのテストがパスすることを確認
- 新機能の場合は対応するテストを追加

### 2. リント
```bash
npm run lint
```
- ESLintエラーがないことを確認
- 警告も可能な限り解消

### 3. TypeScript型チェック
```bash
npx tsc --noEmit
```
- 型エラーがないことを確認

### 4. ビルド確認
```bash
npm run build
```
- ビルドが成功することを確認

### 5. 手動動作確認
- 開発サーバーで実際に動作確認
- エッジケースも含めてテスト

### 6. コード品質
- 不要なconsole.logの削除
- TODO/FIXMEコメントの確認
- パフォーマンスの考慮

### 7. アクセシビリティ
- キーボード操作の確認
- スクリーンリーダー対応
- WCAG 2.1準拠
